# Calculadora

<p>Calculadora creada usando C# con Windows Forms.</p>

<p>Esta aplicación consta de dos pestañas. La primera es la principal que posee los botones de una calculadora, una etiqueta que muestra los resultados de los cálculos y un menú desplegable que lleva a la pestaña de opciones. Permite realizar operaciones de suma, resta, multiplicación, división y resto.</p>
<p>La segunda página es el menú de opciones, el cual da la opción de cambiar el tema y color de fondo de la calculadora. También posee un breve texto en el que se busca incentivar al usuario a dejar su opinion o enviar sugerencias y errores de la calculadora.</p>
